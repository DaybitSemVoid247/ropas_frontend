# Mi Proyecto React + TypeScript + Vite

Aplicación web desarrollada con **React**, **TypeScript** y **Vite**

## Tecnologías usadas

- **React** – Biblioteca principal para la UI
- **TypeScript** – Para tipado estático y mayor seguridad en el código
- **Vite** – Bundler rápido para desarrollo y build
- **TailwindCSS** – Framework de CSS para estilos utilitarios
- **React Router** – Manejo de rutas y navegación en la app
- **React Icons** – Iconos listos para usar
- **ESLint + Prettier** – Para mantener código limpio y consistente

## ¿Cómo ejecutar el código?

- **git clone https://github.com/DaybitSemVoid247/ropas_frontend.git** - para clonar el repositorio
- **cd ropasFrontend** - accede a la carpeta donde esta el proyecto
- **npm install** - instalar las dependencias
- **npm run dev** - iniciar la ejecución del proyecto

## ¿Como ingresar como Administrador?

- **Ambos fueron Entregados por correo o dados por el desarrollador del sistema**
- **correo** - para iniciar como administrador
- **contraseña** - es el formato alfanumerico que se compartio por correo o fue dada por el personal desarrollador del sistema
